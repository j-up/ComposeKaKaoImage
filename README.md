## 모듈
* 클린아키텍쳐 기반으로 모듈 별로 분리되어 있으며 크게 app, data, domain, dataStore로 구분됩니다.

## 화면
* 검색/북마크 탭 전환간 화면 상태가 유지되며 각 아이템에 대한 북마크 정보가 동기화됩니다.

## 기타
* 다국어 처리, 테마 변경 가능합니다.
* Gradle 대신 KTS를 사용하였고 'build-logic' 모듈로 관리됩니다. 
* 각 화면 Preview 작성되어 있어 빌드 전 화면 확인 가능합니다.
